Propósito del sitio

Debido a la creciente demanda de sitios web, apps y ovis virtuales para la educación virtual, en tiempos de pandemia. Venimos a contribuir con nuestro granito de arena a la creciente ola de educación virtual. Este sitio web tiene como objetivo lograr en los estudiantes un aprendizaje optimo sobre la estructura y forma que deben tomar las páginas web mediante el uso de etiquetas semánticas. Y la técnica de CSS Grid Layout que nos permite crear diseños complejos y adaptables con gran facilidad en todos los navegadores.

Etiquetas Semánticas

Son las nuevas etiquetas que vienen incorporadas en la última versión de HTML (HTML5). Pensadas para hacer que la estructura de la página web sea más lógica y funcional. Estas son etiquetas semánticas porque comparten y pertenecen a la misma categoría. En definitiva, en HTML 5 se incorporan las etiquetas semánticas:

  • header: Define la cabecera de la página.
  • footer: Define el pie de página. Contiene datos sobre el creador de la página.
  • nav: Permite definir un elemento de navegación para la página web, puede ser menú principal o menús secundarios.
  • section: Se utiliza para encerrar el código correspondiente a una sección genérica dentro de un documento.
  • aside: Delimita todo aquel contenido que no es directamente contenido principal en concreto. Como bloques publicitarios, enlaces externos, etc.
  • article: Se utiliza para definir artículos como noticias, entradas de un blog, etc. Es probablemente de las más importantes ya que contiene el contenido principal de la web.
Sacado de: Celaya Luna, A. (2014). Creación de páginas web: HTML5. p. 1 - 60.

CSS Grid

Es una técnica que contiene funciones de diseño dirigidas a los desarrolladores de aplicaciones web. Se utiliza para lograr muchos diseños diferentes y adaptables en todos los navegadores. Este modelo de maquetación es en base a una rejilla, permite al autor alinear elementos en columnas y filas.
