Sintaxis y Selectores CSS

La sintaxis y selectores CSS, comprende una serie de sentencias utilizadas para dar estilo y color al contenido de la página web basado en HTML. Un conjunto de reglas CSS consta de un selector y el bloque de declaración, el selector para indicar a que elementos se le aplica el estilo, y el bloque de declaracion contiene una o mas declaraciones que indican como sera el estilo deseado. La declaracion se encierra entre llaves ({...}) y puede contener varias definiciones. Cada definicion consta de un par propiedad/valor, donde el valor esta separado por dos puntos (:), una declaración CSS siempre termina con un punto y coma (;).

Interacción Humano Ordenador

La Interaccion Humano Ordenador o Computador (IHC), es la comunicacion visual y laboral entre la persona y el computador por medio de interfaces y aplicaciones utilizadas por el humano en su computador personal. Esta area es estudiada, evaluada y continuamente mejorada, para lograr una optima interaccion humano-ordenador y facilitar la usabilidad del ordenador.
